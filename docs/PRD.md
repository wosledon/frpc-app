# Frpc App

frpc客户端

**参考配置文件/docs/frpc_full_example.toml**

## 核心需求

1. 支持开机启动
2. 提供自动从github release下载最新的frpc的二进制
3. 支持TCP/UDP/STCP等代理类型
4. 可视化配置/看板
5. 多语言支持

## 功能需求

1. 可以启停frpc服务
2. 配置看板，可以查看各种在工作的代理
3. 可以配置关于frps的配置，例如地址和密钥等需要的配置
4. frpc.toml导入导出

## 页面要求

**使用Material Design进行设计**

1. 首页只需要一个按钮管理是否启动就可以了
